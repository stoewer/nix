NIXAPI size_t nix::util::positionToIndex ( double  position,
const std::string &  unit,
const SetDimension &  dimension 
)

Converts a position given in a unit into an index according to the dimension descriptor.

This function can be used to get the index of e.g. a certain point in time in a Dimension that represents time. The units of the position and that provided by the Dimension must match, i.e. must be scalable versions of the same SI unit.

Parameters
positionThe position
unitThe unit in which the position is given, may be "none"
dimensionThe dimension descriptor for the respective dimension.
Returns
The calculated index.
Exceptions
nix::IncompatibleDimensionThe the dimensions are incompatible.
nix::OutOfBoundsIf the position either too large or too small for the dimension.
